## ğŸ”„ Component Lifecycle

### RoadmapGeneratorModal

```
1. User clicks "Generate with AI"
2. Modal opens (isOpen = true)
3. User types description
4. User clicks "Generate"
5. isGenerating = true
6. Call generateRoadmapFromText()
7. Wait for API response
8. Parse and validate JSON
9. onGenerate(nodes, edges) callback
10. Parent adds nodes/edges to canvas
11. Modal closes
12. Reset state
```

### NodeAIEnhancer

```
1. User clicks AI button on node
2. Component receives current node data
3. User clicks "Enhance with AI"
4. isEnhancing = true
5. Call enhanceNodeContent()
6. Wait for API response
7. Display enhanced preview
8. User reviews changes
9. User clicks "Apply" or "Regenerate"
10. If Apply: onApply(enhancedData)
11. Parent updates node
12. Component closes or resets
```

### NodeAIChatPanel

```
1. User opens Chat tab
2. Component initializes with node context
3. Display welcome message
4. User types message
5. Add to messages array
6. isLoading = true
7. Call startChat() with history
8. Send user message
9. Wait for AI response
10. Add AI response to messages
11. isLoading = false
12. Scroll to bottom
13. Repeat from step 4
```
